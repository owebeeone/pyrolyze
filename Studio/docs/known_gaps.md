# Studio Known Gaps

- Frameless shell exists, but Win32-native move/size system command paths and border-toggle parity are not yet migrated.
- Explorer toolbar actions are present but not yet wired to full folder/refresh/collapse behavior parity.
- Editor tab close/request behaviors and file document model are not yet wired.
- Inspector visual-tree overlay and screenshot annotation remain deferred.
- Persistent settings service exists as contract only; full restore/save behavior pending.
- Rich IDE actions (open/save/edit integrations) remain placeholder by baseline parity intent.
